Mathematical basics

Sets

A set is a collection of objects. If M is a set and if x is an object in this set, we say that x is an element of the set M . Sets consisting of only finitely many elements can be described by the explicit enumeration of their elements. For example, if the set M contains all natural numbers less than 8, we write M = {1, 2, 3, 4, 5, 6, 7}.

Important sets of numbers:

  • Set of natural numbers: ℕ = {1, 2, 3,...}
  • Set of integers: ℤ = {0, 1, -1, 2, -2, 3, -3,...}
  • Set of rational numbers: ℚ
  • Set of real numbers: ℝ

If x is an element of the set M, we write x ∈ M. If x is an object that is not an element of the set M, then we write x ∉ M .

We can also define sets by the properties of their elements. If N is a set and A is a property that some elements in N possess, then
M = {x ∈ N | x possesses property A}
is the set of all elements of N that possess property A.

We say that a set A is a subset of a set B if every element of A is also contained in B. We write this briefly as A ⊆ B. We also call ⊆ an inclusion. If A is a subset of B, we sometimes write B ⊇ A instead of A ⊆ B and we call B a superset of A.

We say that two sets A and B are equal if they contain the same elements, and then write A = B. Moreover, we use ∅ as a symbol for the empty set, i.e., for the set that contains no element. For each object x, x ∉ ∅ and for each set M holds ∅ ⊆ M.

Two sets A and B are equal if and only if A ⊆ B and B ⊆ A holds. To show that two sets are equal, it is often the simplest way to prove these two inclusions separately.

Let M be a set and let A ⊆ M and B ⊆ M be subsets of M. We now define three ways of linking the sets A and B:

  • We denote by A ∪ B the union of A and B, i.e.,
    A ∪ B = {x ∈ M | x ∈ A or x ∈ B}.
  • We denote by A ∩ B the average of A and B, i.e.,
    A ∩ B = {x ∈ M | x ∈ A and x ∈ B}.
  • We denote by A \ B the difference of A and B, i.e.,
    A \ B = {x ∈ M | x ∈ A and x ∉ B}.

Two sets A and B with A ∩ B = ∅ we call disjoint.

A set that contains only a finite number of elements we call a finite set. For a finite set M we denote by |M| its cardinality, i.e. the number of elements it contains.

It is easy to consider that for any two finite sets A and B the following equation always holds:
|A ∪ B| = |A| + |B| - |A ∩ B|.

Another way to combine two sets A and B is the Cartesian product A × B. The elements of A × B are all ordered pairs (a, b) with a ∈ A and b ∈ B, i.e.
A × B = {(a, b) | a ∈ A and b ∈ B}.
We call these pairs ordered because the order of the elements is important and in general (a, b) != (b, a) holds. One should not confuse the pair (a, b) with the set {a, b} = {b, a}.

For a set M we denote by P(M) its power set. This is the set of all subsets of M , i.e.
P(M) = {X | X ⊆ M}.

Proofs

A mathematical proof is about proving the correctness of a statement. One starts with a set of basic statements, which one assumes to be true. These are the so-called axioms. In a direct proof the correctness of further statements is proven from these axioms by logical conclusions, until one arrives at the statement to be proven. In an indirect proof, one assumes that the statement to be proven is false and derives a contradiction from this assumption.

Statements

By a statement we first understand quite generally propositions that can be true or false. In contrast to everyday statements, we are interested in statements that have a definite truth value.

We usually denote statements by capital letters A, B, C,... and for a statement A we denote by w(A) its truth value. Here, w(A) = 0 means that the statement A is false, and w(A) = 1 means that it is true.

If A is a true statement, we say that A holds. If, on the other hand, A is a false statement, we say that A does not hold.

Statements only become interesting when you can link them.
If A and B are two statements, then A ∧ B is also a statement, which we call the conjunction of A and B. The conjunction is true if both statements A and B are true. It is true exactly if both statements A and B are true.
Also A ∨ B is a statement we call the disjunction of A and B. It is true exactly if at least one of the statements A and B is true.
For a statement A, we denote by ¬A its negation. This statement is true exactly if A is false.

We can also describe a linkage by its truth table. In such a table, for each possible combination of truth values, the truth value of the linkage is shown.

A ¬A
0 1
1 0

A B A ∧ B

1 0 0
